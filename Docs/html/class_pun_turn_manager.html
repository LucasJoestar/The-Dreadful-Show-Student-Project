Detailed Description

Pun turnBased Game manager.

Provides an Interface (IPunTurnManagerCallbacks) for the typical turn flow and logic, between players Provides Extensions for PhotonPlayer, Room and RoomInfo to feature dedicated api for TurnBased Needs
